华南理工大学五山校区周边餐饮分析

2026.06· 数据分析 Python · 高德API · DeepSeek · Folium · Pandas · Matplotlib

本项目从奶茶店爬取作业升级为完整餐饮数据流水线。以华南理工大学五山校区为中心、2000m 为半径，通过高德地图 API 5×5 网格化采集全品类餐饮 POI，调用 DeepSeek 大语言模型将店铺自动分类为 10 个类别（火锅、烧烤、茶饮咖啡、粉面小吃、中餐正餐、异国料理等），含边界案例 few-shot，最终生成 Folium 交互式地图与 Matplotlib 统计图表。

核心特性

  • 25 格网格化采集，覆盖 2000m 半径
  • API 限流自动重试（指数退避）
  • DeepSeek 批量分类 + 结果缓存
  • LayerControl 10 类别 + 热力图自由切换
  • MarkerCluster 聚合标记 + 高德导航弹窗
  • 右上角统计图例面板 + 采集日期

技术栈

  • 高德地图 Web API（around 接口）
  • DeepSeek Chat API（few-shot 分类）
  • Folium + Leaflet（交互地图）
  • Matplotlib + Seaborn（统计图表）
  • Pandas（数据处理与清洗）
  • python-dotenv（API 密钥管理）

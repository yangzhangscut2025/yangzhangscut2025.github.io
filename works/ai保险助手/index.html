AI保险规划助手

2024.03 – 2024.05· AI与数据应用 Flask · 通义千问 · RAG · ChromaDB

项目背景

以 Z 世代为目标用户，打造游戏化 AI 保险顾问。集成阿里云通义千问大模型与保险知识库，实现趣味问答与智能推荐。项目获保险AI创新黑客松大赛一等奖。

核心功能

  • 趣味问答 — AI 扮演保险推荐助手，逐步引导用户提供健康信息、收入情况，生成个性化保险方案推荐。支持上传 PDF 保险方案链接，AI 自动解析并总结。
  • 角色扮演模拟 — 用户扮演金牌保险销售，AI 扮演有保险需求的客户角色。销售需解答客户顾虑提升购买意向值，支持自定义角色与 TTS 语音互动。
  • 混合 RAG 知识库 — 采用本地检索 + 云端生成的混合架构。保险文档本地分块 → BGE 嵌入模型 → ChromaDB 向量库，检索结果拼装 prompt 后调用云端大模型生成答案，文档不上云。

技术架构

后端框架Flask 3.0 + Flask-Login + Flask-SQLAlchemy
大模型阿里云通义千问（Qwen-max / Qwen-plus）
RAG 向量库ChromaDB + BAAI/bge-small-zh-v1.5（本地嵌入）
语音合成阿里云 DashScope TTS（8 种音色）
前端HTML + CSS + JS（jQuery + Bootstrap 4）
